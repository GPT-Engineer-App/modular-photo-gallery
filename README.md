# modular-photo-gallery

A photo site with widgetized elements, including the title section. The site should feature a modular approach to content display, allowing for easy updates and customization of photographic elements and the title. The interface should be intuitive and visually appealing, catering to both desktop and mobile users.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/modular-photo-gallery.git
cd modular-photo-gallery
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
